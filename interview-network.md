### Netty — 一个 Java 开源框架。Netty 提供异步的、事件驱动的网络应用程序框架和工具，用以快速开发高性能、高可靠性的网络服务器和客户端程序
- Netty


### Apache Mina — 一个网络应用程序框架，可帮助用户轻松开发高性能和高可伸缩性的网络应用程序。它通过 Java NIO 通过各种传输（如：TCP/IP 和 UDP/IP）提供了一个抽象的事件驱动的异步 API
- Mina
